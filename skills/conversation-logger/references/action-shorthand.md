# Action Shorthand Reference

Quick reference for documenting actions in the log.

## Tool Actions

```
Read {filename}
Created {filename}
Updated {filename}
Deleted {filename}
Glob: found N files matching {pattern}
WebSearch: {query}
Bash: {command purpose}
```

## Response Types

```
Answered question about {topic}
Explained {concept}
Asked N clarifying questions
Provided code for {purpose}
Listed N options for {decision}
```

## Status Markers

| Marker | Meaning |
|--------|---------|
| ✅ | Complete |
| 🚧 | In Progress |
| ❌ | Blocked |
| 📋 | Planning |
| 👀 | Needs Review |
